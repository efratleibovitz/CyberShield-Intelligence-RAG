Cyber Shield - Groq Direct Mode

What are the global development instructions mentioned in the files?

The global development instructions mentioned are:

  1. Critical Alerts:

    • Every 'Critical' severity alert must trigger an SMS to the Security Manager.
    • Display critical alerts in #FF0000 (Bright Red) and warning/other alerts in #FFA500 (Orange) on the UI.

  2. Code Style:

    • Use asynchronous functions for all network scanning tasks.

  3. Networking:

    • Use a distributed server architecture.
    • Allow only Port 443 (HTTPS) and block Port 80 (HTTP) automatically.

  4. Storage:

    • Store logs in an S3 Bucket for 90 days.

What are the specifications for the Cyber-Shield system?

The Cyber-Shield system specifications include:

  1. Networking:

    • Environment: Distributed server architecture.
    • Firewall Config:
      • Allow Port: 443 (HTTPS) only.
      • Block Port: 80 (HTTP) automatically.

  2. Storage:

    • Logs are stored in an S3 Bucket for 90 days.

Additionally, the system should adhere to the provided global development instructions, including critical alerts and code style guidelines.
